# Changelog
All notable changes to Blender Texture Optimizer (BTO) will be documented in this file.

## [1.0.0] - Initial Release
### Added
- Scene Texture Scanner
- VRAM usage estimation
- Oversized texture detection
- Packed texture detection
- Missing texture detection
- Duplicate texture analysis
- Optimization recommendation system
- Safety-tier optimization workflow:
  - Safe
  - Balanced
  - Review
- Batch texture downscaling
- Batch texture conversion
- Optimization presets:
  - VRChat Quest
  - Mobile
  - Game Ready
  - Portfolio Render
  - Web Optimized
- Before vs After analytics dashboard
- Non-destructive optimization workflow
- Native Blender sidebar integration
- Texture optimization queue system
